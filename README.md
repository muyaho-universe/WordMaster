### 현재까지 프로젝트를 구현하면서 소감, 어려움, 또는 건의사항

(2022년 9월 4일) </br>
오랜만에 다루는 자바인 만큼 클래스를 나누거나 인터페이스를 구축하는 등의 여러 기본적인 작업에서 어려움이 있었다. </br>
그래도 교수님께서 올려주신 강의와 다시 연습을 해보니 손에 익었다. 이번 학기 실전프로젝트 수업을 통해 귀중한 프로젝트 경험을 쌓길 바란다.

(2022년 9월 16일) </br>
자바를 통하여 CRUD를 오랜만에 구현해 보았다. 클래스를 나눈다던지, 인터페이스를 쓴다는 것이 쉽지는 않았지만, 내가 무엇을 만들어야 하는지 고민을 하며 하다보니 쉽게 마무리를 할 수 있었던 거 같다. </br>
그럼에도 교수님께서 알려주신 자료에서 크게 벗어나지 못했다라는 점은 큰 아쉬움으로 남는다.</br> 
더욱 정진하여 나만의 프로젝트를 만들어야겠다고 느꼈다.
